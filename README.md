# Sora
Transporte exclusivo para mujeres

Para levantar el proyecto:

1 - npm install
2 - npm run dev

Carpeta API

1 - Crear una DB con el nombre 'sora'
2 - Crear un archivo .env en la carpeta api
    DB_USER=postgres user
    DB_PASSWORD=password postgress
    DB_HOST=localhost   
3 - npm install
4 - npm start
